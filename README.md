# Complaint System

##  Description
This is a simple web application where users can submit complaints through a form and view all submitted complaints.

## Features
- Submit complaints using a form  
- Display complaints in a table  
- Data remains saved even after refreshing  
- Simple and easy-to-use interface  

##  How It Works
The application stores complaints in the browser and shows them on the page after submission.

##  Technologies
- HTML  
- CSS  
- JavaScript
